Internet

Internet is a global network that allows people to communicate and receive information from all over the world through devices, like smartphones and computers.

Web

Web is all the public websites and pages that can be accessed the users from their devices through internet.

How are the Internet and Web different?

Internet is the network that allows everyone to access so people can communicate globally, while web is the specific information can be accessed through the internet.

ASCII and Unicode

ASCII and Unicode are standards code that used as a represent characters and text on devices, while ASCII has limited characters and Unicode has wider range of characters.

UTF-8

UTF-8 is an encoding system that acts for characters in size of 1 - 4 bytes.

URL

URL (Uniform Resource Locator) is the address of sources on the internet.

Domain Name

Domain Name is a unique name that associated with IP address on the internet.

HTTP

HTTP (Hypertext Transfer Protocol) is an application protocol for distributed, collaborative, hypermedia information systems that allows users to communicate data on the internet.

HTTPS

HTTPS is an extention or the secure version of HTTP with the primary protocol is to send data between a browser and a website.

IP Address

IP Address (Internet Protocol Address) is the unique a numeric label assigned to connect every device to the internet.

DNS

DNS (Domain Name System) is a system that turns the domain name into IP address, which allow browsers to get to the websites and other resources.

W3C Recommendations

World Wide Web Consortium Recommendation is standards of guidelines that are used to promote consistency and interoperability through the internet.

XML, XHTML, HTML, HTML5

XML is a tool for storing and transporting data, could be a software and hardware. XHTML is part of the XML markup languages that used for creating and showing web pages, with more lenient in its syntax. HTML is the standard markup language for designing documents that will be displayed in a web browser. HTML5 is the latest version of HTML in markup language that used to develop web hypertext documents on the World Wide Web.
