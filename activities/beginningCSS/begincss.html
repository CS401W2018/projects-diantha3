1. Explain the difference between inline, embedded, and external CSS.
Inline is one of the styles in HTML that is used as part of the HTML tag itself. The application of this style is using style attribute that directly within an HTML tag. Embedded style is placed in the header of the page and applied to the specific of that page entire. And the external style is used in a separate file and directly linked to the HTML document. The application of this style is using the link tag.

2. Explain why you should only use external CSS.
Because this style is the best application for most web development projects, especially for building larger website. This style is also ideal for styling multiple pages with the same style sheet.

3. What is a selector used for?
A selector is used for selecting which the HTML elements on the webpage that we ant to style. A selector also allows us to apply the color, font size, layout, etc. to elements based on the attributes.
