1. Block tags refers to an element that starts on a new line and have a height, widht depend on the text. While inline tags don't have a height and width because they don't start on a new line, only occupies the space that required by the contents itself.

2. Margin is the space outside of the box or the element's border, padding is the space inside of the box or between an element's content and its border, then border is the (visible) line out of the box or line that surround the content.

3. For the fixed size we can use px (pixel), thin, in, cm; also relative size can use % (percentage) and em.

4. We use the figure tags when we want to embed a content like a picture with caption on it.
