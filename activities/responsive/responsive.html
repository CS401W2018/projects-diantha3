1. Why is the viewport meta tag important?
Because it helps ensure that a website displays properly across different devices. Without it, mobile devices might display pages too wide, making them difficult to read.

2. How can you adjust the CSS for printing?
Make new file (ex: "print.css"), then copy paste all the modify from css to the new file named "print.css"

3. When would you want to adjust the CSS for printing?
When we want to print the content from a webpage.
