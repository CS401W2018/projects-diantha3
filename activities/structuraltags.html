Structural Tags

1. What goes in the header?
Header usually contain the section's headings or title also used to wrap a section's list of the content.

2. What goes in the Nav? Can the Nav go in the header? Does it have to go in the header? Does the Nav contain links to another website?
Nav stands for navigation, which is to help the user easier to find what are they looking for on the page.Yes, nav can go in the header but nav doesn't really have to be in the heading and usually nav links to other sources but not other website.

3. What goes in main?
Main contains the main topic or content on the webpage.

4. What goes in the Footer?
Footer contains the copyright of the webpage and usually more other information that will be more helpful for the user.

5. HTML Validation
HTML Validation is a process to test is the HTML code has an error or not and make sure that the code is working.
