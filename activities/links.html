Explain the difference between an absolute link and a relative link. Why would you use relative links?
The main difference absolute and relative links stand on how they specify the location of a resource, like webpage. Absolute link used to direct link to outside of the website, while relative link used to direct link inside or from the same website.
Because, if I still at the same website or domain, relative link is needed.

List the 3 rules with relative links

  • Should use the file name
  • Use double dot(..) if want to change the directory (go up 1 directory/folder)
  • Always start at the root if link starts with "/"
